# 💰 FinTrack AI – Personal Finance & Expense Tracker

🚀 A modern full-stack finance management platform that helps users track expenses, manage budgets, visualize spending patterns, and receive AI-powered financial insights for smarter money management.

---

## 🌟 Features

### 🔐 Secure Authentication

* User Registration & Login
* JWT-based Authentication
* Password Encryption using bcrypt
* Protected Routes

### 💳 Expense Management

* Add, Edit, and Delete Transactions
* Categorize Income and Expenses
* Real-time Transaction Tracking
* Transaction History Management

### 📊 Interactive Dashboard

* Financial Overview
* Total Income & Expenses
* Balance Tracking
* Quick Financial Summary

### 🎯 Budget Planning

* Create Monthly Budgets
* Monitor Spending Limits
* Budget Utilization Tracking
* Overspending Alerts

### 📈 Reports & Analytics

* Expense Analysis
* Spending Trends
* Category-wise Breakdown
* Financial Performance Insights

### 🤖 AI Financial Insights

* Smart Spending Analysis
* Personalized Financial Recommendations
* Expense Pattern Detection
* AI-Powered Budget Suggestions

## ⚙️ Installation & Setup

### Clone Repository

```bash
git clone https://github.com/Jawa-vit/Personal-Finance-Expense-Tracker-with-AI-Insights.git
cd Personal-Finance-Expense-Tracker-with-AI-Insights
```

### Backend Setup

```bash
cd backend
npm install
```

Create a `.env` file:

```env
PORT=5000
MONGO_URI=your_mongodb_connection_string
JWT_SECRET=your_secret_key
CLAUDE_API_KEY=your_api_key
```

Run Backend:

```bash
npm run dev
```

### Frontend Setup

```bash
cd frontend
npm install
npm run dev
```

Application will run at:

```text
Frontend : http://localhost:5173
Backend  : http://localhost:5000
```
